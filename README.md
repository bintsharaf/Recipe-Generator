# AI Recipe Generator

## Project Overview
A lightweight web application that allows users to generate cooking instructions instantly by entering a recipe name. The app reduces recipe search time by up to **70%** and provides a clean, mobile-first interface.

This project demonstrates **DOM manipulation, responsive UI design, and interactive user interfaces**.

---

## Features
- Input a recipe name to generate cooking instructions.  
- Responsive design for mobile and desktop.  
- Interactive UI for a smooth user experience.  

---

## Tech Stack
- HTML5  
- CSS3  
- JavaScript (ES6+)
